# 双色球历史数据规律分析

一个基于Streamlit的双色球历史数据可视化分析工具，帮助用户发现号码规律，辅助决策。

## 功能特点

### 📊 基本数据概览
- 显示数据统计信息
- 查看最新开奖结果
- 数据导出功能（CSV/Excel）

### 🔴 红球号码分析
- 红球出现频率分布
- 红球号码热力图
- 红球区间分布
- 红球出现频率TOP10
- 最近未出现的红球

### 🔵 蓝球号码分析
- 蓝球出现频率分布
- 蓝球奇偶分布
- 蓝球大小分布
- 蓝球走势折线图
- 蓝球出现频率TOP5
- 最近未出现的蓝球

### 🎯 号码组合分析
- 红球奇偶比分析
- 红球大小比分析
- 红球连号分析
- 红球和值分析
- 红球跨度分析

### 📈 历史趋势分析
- 奖池金额趋势
- 红球和值趋势
- 蓝球大小趋势
- 红球奇偶趋势
- 红球区间趋势
- 红球号码热度趋势

### 🤖 智能号码推荐
- 基于历史数据分析的智能推荐
- 可调整热门/冷门号码权重
- 生成多组推荐组合
- 提供推荐依据说明

## 安装与运行

### 环境要求
- Python 3.8+
- pip

### 安装步骤

1. 克隆或下载本项目
2. 安装依赖包
```bash
pip install -r requirements.txt
```

3. 运行应用
```bash
streamlit run app.py
```

## 使用说明

1. **数据管理**
   - 应用启动时会加载内置的历史数据
   - 点击"更新最新数据"按钮可尝试从网络获取最新开奖结果

2. **数据范围选择**
   - 可选择"全部数据"、"最近50期"、"最近100期"、"最近200期"或"自定义范围"
   - 自定义范围可通过日期选择器设置具体的起止日期

3. **分析功能**
   - 在左侧导航栏选择需要的分析功能
   - 每个功能模块都有详细的图表和数据展示
   - 部分功能支持参数调整，如移动平均线窗口大小、推荐权重等

4. **数据导出**
   - 在"基本数据概览"模块可导出当前筛选的数据为CSV或Excel格式

## 数据说明

- 初始数据包含从2025051期至2026018期的双色球历史开奖数据
- 数据来源：网络公开数据
- 每次启动应用时会尝试从网络获取最新数据，如获取失败则使用本地数据

## 重要提示

- 本应用仅供学习和娱乐使用，不构成购彩建议
- 彩票中奖号码是随机产生的，历史规律不代表未来趋势
- 请理性购彩，控制购彩金额，享受彩票带来的乐趣

## 技术栈

- **前端框架**：Streamlit
- **数据处理**：Pandas, NumPy
- **数据可视化**：Matplotlib, Seaborn
- **网络爬虫**：Requests, BeautifulSoup
